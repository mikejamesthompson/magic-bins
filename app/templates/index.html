What is this site?

This website allows residents of Sevenoaks District Council to find out when their next bin collection is going to be, even over public holidays when things get a bit weird. It was created because the current ways of finding out are a bit limited.

How does it work?

This site is based on data extracted from this file on Sevenoaks DC's website and from information on this page of the site. As far as I know, it is correct, but there may be areas. Get in touch if you spot any and remember that Sevenoaks DC are the final arbiters of this information.
